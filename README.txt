# SARAH-Plugin-tvneufbox
Plugin tvneufbox version 1.0 pour SARAH by DUPONT Amaury

Participation de Jérôme VEYRET et Benoit GUILLAUME pour la partie plugin (Merci à eux pour leur aide)
Participation de "Insedo" de TDM pour la partie pilotage http.


***********************************************************************************************************
Préambule:

Afin que le plugin puisse fonctionner, il faut avant tout installer un petit script écrit par Insedo de TLD 
sur un serveur PHP5 minimum (Dossier stb).
Un tuto trés simple et clair permet de suivre la procédure sans problème:

http://www.touteladomotique.com/forum/viewtopic.php?f=42&t=8088&p=73980&hilit=neufbox&sid=a48f800f5b289ba3ce7db13f31a1208d#p73980

***********************************************************************************************************
Partie Plugin SARAH:

Une fois la partie script installée, il suffit d'intaller le plugin SARAH, d'y configurer l'adresse du
serveur via le portail... and Enjoy ;-)
Toutes les commandes vocales sont à entrer dans le fichier XML.
Dans ce plugin, les actions principales y ont été créé, mais vous pouvez configurer suivant vos besoin
et univers SFR de votre abonnement.

NB: Concernant les différentes chaines, il est préférable d'écrire les commandes vocales en version phonétique
pour éviter toute erreur de compréhension de SARAH.
